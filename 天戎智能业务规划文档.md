## 公司简介与定位

### 公司定位

**天戎智能**是一家专注于巡检场景的高度定制化解决方案提供商，致力于为物流仓储、公共安全、工业园区等领域提供智能化巡检服务。

### 核心能力

天戎智能的核心能力涵盖三大领域：

- **云控平台（RSP）**：机器人远程控制与遥操作平台，实现从远程操控到自主巡检的全链路解决方案
- **设备传感器集成**：硬件背包设计与系统集成，提供外观壳体设计、模块安装、散热等一体化解决方案
- **数据采集平台**：独立的数据采集与分析平台，形成巡检业务闭环

### 商业模式

## 核心产品与技术体系

### 产品矩阵

天戎智能的产品体系由三大核心产品组成：

#### 1. RSP 云控平台（遥操平台）

RSP（Robot Service Platform）是天戎智能的核心产品，提供从远程操控到自主巡检的全链路解决方案。

**版本演进路线**：

| 版本 | 能力等级 | 核心功能 | 应用场景 |
|------|----------|----------|----------|
| **V1.0.0** | 遥控级 | 遥控、巡检、视频流、对讲 | 室内场景、物流仓储 |
| **V1.1.0** | 半自主级 | 360°遥控、热成像预警、对讲、警报、拍照 | 复杂场景、安全巡检 |
| **V1.2.0** | 全自主级 | 基于V1.1.0 + 自主导航 + AI视觉识别 | 全场景、智能巡检 |

**技术架构**：

```
┌────────────────────────────────────────────────┐
│              RSP 云控平台架构                   │
├────────────────────────────────────────────────┤
│                                                │
│  ┌──────────┐    ┌──────────┐    ┌──────────┐ │
│  │ 遥控模块 │    │ 巡检模块 │    │ 对讲模块 │ │
│  └──────────┘    └──────────┘    └──────────┘ │
│                                                │
│  ┌──────────┐    ┌──────────┐    ┌──────────┐ │
│  │ 视频流   │    │ 热成像   │    │ AI视觉   │ │
│  └──────────┘    └──────────┘    └──────────┘ │
│                                                │
│  ┌──────────────────────────────────────────┐ │
│  │         自主导航系统                      │ │
│  │  激光SLAM → 视觉融合SLAM（未来升级）      │ │
│  └──────────────────────────────────────────┘ │
│                                                │
└────────────────────────────────────────────────┘
```

#### 2. 数采平台（独立平台）

数采平台是一个独立的数据采集与分析平台，与 RSP 云控平台形成业务闭环。

**核心功能**：
- 实时数据采集与存储
- 数据分析与可视化
- 异常预警与报告生成
- 历史数据回溯与分析

**业务闭环**：

```
巡检执行 → 数据采集 → 分析处理 → 决策反馈 → 优化巡检
    ↑                                           │
    └───────────────────────────────────────────┘
```

#### 3. 硬件背包

硬件背包是天戎智能的核心集成产品，将多个传感器集成到一个统一的背包模块中。

**设计内容**：
- 外观壳体设计
- 模块安装设计
- 散热系统设计
- 传感器集成（摄像头、热成像、激光雷达等）

**合作本体**：

天戎智能基于多家机器狗厂商的本体产品，提供定制化的传感器集成方案：

| 厂商 | 产品型号 | 特点 |
|------|----------|------|
| **智身科技** | zsl-1、zsl-1w、M1 | 官方 EDU 背包方案（含多传感器集成） |
| **云深处** | M20、S10 | 高性能机器狗本体 |

### 技术体系

#### 导航技术路线

天戎智能采用两套 SLAM（同步定位与地图构建）技术方案：

**1. 激光 SLAM（当前方案）**

- 激光定位
- 全局路径规划
- 实时避障
- 环境建图

**2. 视觉融合 SLAM（未来升级）**

- 视觉 + 激光融合定位
- 更强的环境适应性
- 全局路径规划
- 实时避障
- 环境建图

**技术路线图**：

```
当前阶段               未来阶段
    │                     │
    ▼                     ▼
激光SLAM    ────────▶  视觉融合SLAM
(成熟稳定)              (升级替代)
```

#### 狗窝系统（充电站）

狗窝是巡检体系的重要一环，为机器狗提供自动充电和数据同步功能。

**核心模块**：
- 配电盒模块
- 温湿度传感器
- 通讯模块
- 数据采集模块

## 业务场景与解决方案

### 应用场景

天戎智能的巡检解决方案适用于多种场景：

#### 1. 物流仓储巡检

**典型案例**：普罗斯物流试点项目

**项目背景**：
- 国内物流仓储场景试点
- 采用 RSP V1.0.0 平台
- 实现遥控巡检、视频监控、远程对讲

**解决方案**：
- 机器狗本体 + 硬件背包（传感器集成）
- RSP 云控平台（遥控级）
- 数采平台（数据闭环）

**价值**：
- 降低人工巡检成本
- 实时监控仓库环境
- 快速响应异常情况

#### 2. 公共安全巡检

**典型案例**：美国警局试点项目

**项目背景**：
- 美国市场试点项目
- 局部遥控器 APP 开发
- 结合 hawkrboo 闭环系统

**解决方案**：
- 局部通讯方案
- 5G 通讯模块
- 定制化硬件设计（zsl-1 背板、外形定制）

**价值**：
- 提升警务巡检效率
- 降低人员安全风险
- 实时数据回传与分析

#### 3. 停车场巡检

**典型案例**：西雅图停车场项目

**项目特点**：
- 新项目，不集成 RSP
- 基于独立遥控器方案
- 基于 V1.2.0 技术栈

#### 4. 工业园区巡检

**应用场景**：
- 彩钢房巡检
- 配电箱巡检
- 局部通讯场景

**解决方案**：
- 模块集成方案（局部通讯）
- 现有模组集成（ro-box、orin、继电器、对讲、散热）
